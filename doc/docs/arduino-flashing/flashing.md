# Flashing the Arduino

1. Connect the arduino using the USB port
> ⚠️ Never use the USB port whilst the DC power jack connected
2. In the Arduino IDE select `Tools > Board > Arduino Leonardo`
3. Then from `Tools > Port` select the `(Arduino Leonardo)` option
4. Open the `midi-stomptroller.ino` sketch and make changes if necessary
5. Press the `Upload` arrow in the top left
